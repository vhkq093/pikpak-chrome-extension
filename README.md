## PikPak 是什麼？先搞清楚它跟一般雲端硬碟有什麼不同

簡單說，PikPak 是一個「雲端離線下載器 + 影音播放器」，而不是你印象中那種把本地檔案丟上去的雲端硬碟。

它最核心的邏輯是：**你給它一個連結（磁力、BT 種子、HTTP 直連，甚至 TikTok、X/Twitter 上的影片分享連結），PikPak 自己跑去下載，存進你的私密雲盤，你再從任何裝置去看。** 整個下載過程在他們的伺服器上完成，你的本地頻寬完全不受影響。

這個設計思路跟傳統的 qBittorrent 之類的本地 BT 下載工具完全不同。用 PikPak，你不用擔心自己的 IP 被記錄，下載速度也不受家裡頻寬限制，只要 PikPak 的伺服器端抓得到，就算是冷門資源也常常可以秒存。

## PikPak Chrome 擴充功能到底能做什麼

Chrome 擴充功能（PikPak Browser Extension）是 PikPak 整個生態系統裡最方便的入口之一。裝上去之後，主要帶來以下幾個變化：

**1. 自動偵測並顯示一鍵提交按鈕**

安裝後，頁面上所有的磁力連結和 BT 資源旁邊，會自動顯示一個 PikPak 圖示按鈕。點一下，該連結就直接提交至你的 PikPak 雲盤開始下載，整個過程不需要複製連結、切換視窗，非常流暢。

**2. 右鍵選單支援「儲存到 PikPak」**

如果你覺得圖示按鈕有點礙眼，可以在設定裡把它關掉。改用右鍵點擊連結，選單裡就有「儲存到 PikPak」的選項，用起來更低調一些。

**3. 彈出視窗可直接貼上任意連結**

不只是頁面上現成的連結，點開擴充功能的彈出視窗，也可以手動貼上磁力連結或其他資源網址直接提交。

**4. 顯示時機可調整**

如果你不希望圖示一直出現打擾瀏覽，可以在基本設定裡把它改成「滑鼠懸浮時顯示」或「選中連結時顯示」，按需使用。

**Q：PikPak Chrome 擴充是官方出的嗎？**

**Q：免費版能不能用擴充功能？**

可以，但每日提交任務次數有限制。如果你只是偶爾用，免費版夠了；如果是每天都要抓東西，建議升級 Premium。

**Q：全球會員和區域會員怎麼選？**

PikPak 會自動根據你的 IP 和裝置資訊判斷，通常不需要你自己選。如果你在台灣、香港、日本、南韓這些地方，系統可能判定為全球會員，價格會比區域會員高一些，但速度上限也更高。
